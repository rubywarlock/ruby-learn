Паттерн `Factory` (Фабрика) относится к порождающим паттернам проектирования  
и используется для создания объектов без предварительного знания их конкретного класса.  
Фабрика абстрагирует процесс создания объектов,  
обеспечивая клиенту интерфейс для создания экземпляров объектов с определенными свойствами.  
  

Паттерн `Factory` обеспечивает гибкость и позволяет создавать объекты с определенными свойствами и поведением,  
используя общий интерфейс, предоставляемый фабрикой.
